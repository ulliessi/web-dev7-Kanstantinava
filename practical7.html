Repaint and Reflow

Repaint involves re-rendering affected pixels without changing layout, while reflow recalculates layout of affected elements.

JavaScript and Page Performance

JavaScript can impact page load speed by delaying rendering, causing reflows, or blocking the main thread. It can also enhance performance through asynchronous operations and lazy loading.

Minification of Scripts

Minification removes unnecessary characters from scripts, reducing file size and improving page load times. It's useful for optimizing bandwidth usage and enhancing website performance.

JS Web Worker

A JavaScript Web Worker runs scripts in the background, separate from the main thread. It's used for CPU-intensive tasks, enabling responsive user interfaces while performing complex operations.
